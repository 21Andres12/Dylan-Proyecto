Caso 1: Programación Por Metas (Panadería La Espiga Dorada S.A.)

Contextualización del Problema

El proyecto analiza la situación operativa de Panadería La Espiga Dorada S.A., ubicada en Grecia, Alajuela. La empresa produce dos líneas: pan artesanal (alta rotación) y repostería fina (mayor margen, laborioso). Se debe planificar la producción semanal enfrentando tres metas fundamentales en orden estricto de importancia (Prioridades):

  • Prioridad 1 (Rentabilidad): Alcanzar una ganancia semanal de exactamente $2200 para cubrir el punto de equilibrio.
  • Prioridad 2 (Acuerdo laboral): Asignar exactamente 160 horas de trabajo semanales (ni más ni menos).
  • Prioridad 3 (Política ambiental): Limitar el consumo de energía a un máximo de 300 kWh semanales.

Estructura de las Metas

Prioridad Meta Objetivo Desviación
P1Rentabilidad$2200$d_1^-$ (Faltante)
P2Horas Laborales160 hrs$d_2^-, d_2^+$ (Ambas)
P3Energía Horno300 kWh$d_3^+$ (Exceso)

Consumo de Recursos por Lote

Producto Ganancia Trabajo Energía
Pan ($x$)$302 horas5 kWh
Repostería ($y$)$504 horas8 kWh

Desarrollo Matemático Paso a Paso

Se plantean las ecuaciones agregando las variables de desviación adicionales ($d_i^-$ y $d_i^+$):

Ecuación de Meta 1 (Ganancia): $30x + 50y + d_1^- - d_1^+ = 2200$

Ecuación de Meta 2 (Trabajo): $2x + 4y + d_2^- - d_2^+ = 160$

Ecuación de Meta 3 (Energía): $5x + 8y + d_3^- - d_3^+ = 300$

La función objetivo busca minimizar las desviaciones indeseadas según su nivel de prioridad:

$$\text{Minimizar } Z = P_1(d_1^-) + P_2(d_2^- + d_2+) + P_3(d_3^+)$$

Resolviendo el sistema base para cumplir de forma estricta las metas de alta prioridad P1 y P2 ($d_1^- = 0, d_2^- = 0, d_2^+ = 0$):

$3x + 5y = 220 \quad \text{(Ganancia simplificada)}$

$x + 2y = 80 \implies x = 80 - 2y \quad \text{(Trabajo simplificado)}$

Sustituyendo: $3(80 - 2y) + 5y = 220 \implies 240 - y = 220 \implies y = 20$

Calculando $x$: $x = 80 - 2(20) \implies x = 40$

Evaluación Final de Logros:
Lotes de Pan ($x$)40
Lotes Repostería ($y$)20
Exceso Energía ($d_3^+$)60 kWh

Interpretación: Las metas P1 y P2 se satisfacen al 100%. Sin embargo, para lograrlo se requiere consumir $5(40) + 8(20) = 360 \text{ kWh}$, lo que genera un exceso inevitable de 60 kWh sobre la meta ambiental (P3).

Caso 2: Programación Entera Pura (Planificación Taller Textil)

Enunciado del Caso

Un taller textil confecciona camisas ($x_1$) y pantalones ($x_2$). Cada prenda requiere insumos específicos y deja una utilidad unitaria definida. Al fabricarse unidades completas, las variables deben ser estrictamente enteras.

Modelo de Optimización

$$\text{Maximizar } Z = 60x_1 + 50x_2$$

Sujeto a:

$x_1 + 2x_2 \le 10$ (Restricción de Tela)

$2x_1 + x_2 \le 11$ (Restricción de Tiempo)

$x_1 + x_2 \le 9 \quad$ (Restricción de Botones)

Puntos Límite de Rectas

• Tela ($R_1$): Corta en $(0, 5)$ y $(10, 0)$

• Tiempo ($R_2$): Corta en $(0, 11)$ y $(5.5, 0)$

• Botones ($R_3$): Corta en $(0, 9)$ y $(9, 0)$

Evaluación de Vértices de la Región Factible

Vértice Factible $(x_1, x_2)$ Cálculo de Utilidad ($Z$) Resultado Total
$(0, 0)$$60(0) + 50(0)$$0
$(5.5, 0)$$60(5.5) + 50(0)$$330
$(0, 5)$$60(0) + 50(5)$$250
$(4, 3)$$60(4) + 50(3) = 240 + 150$$390
Plan de Confección Óptimo Semanal:
Camisas a Fabricar4 unidades
Pantalones a Fabricar3 unidades
Utilidad Máxima$390

Nota Operativa: La intersección de las limitantes de tela y tiempo ocurre exactamente en el punto entero $(4,3)$, el cual valida la condición de botones ($4+3 = 7 \le 9$). Al ser coordenadas enteras, la solución es directa sin requerir ramificación.

Caso 3: Programación No Lineal (Maximización de Beneficios)

Estructuración de Utilidades Cuadráticas

Una fábrica vende productos A y B cuyas demandas son elásticas, modelando ingresos no lineales. Restando sus costos de producción lineales ($5x$ y $8y$) a las funciones de ingresos, se construye la función de beneficio neto $Z$:

$$Z(x,y) = (40x - x^2 - 5x) + (50y - 2y^2 - 8y)$$

$$Z(x,y) = 35x - x^2 + 42y - 2y^2$$

Optimización Mediante Cálculo Diferencial

Se extraen las primeras derivadas parciales (Gradiente) para hallar las coordenadas del punto crítico global del beneficio:

$$\frac{\partial Z}{\partial x} = 35 - 2x = 0 \implies x = \frac{35}{2} = 17.5$$

$$\frac{\partial Z}{\partial y} = 42 - 4y = 0 \implies y = \frac{42}{4} = 10.5$$

Prueba de Factibilidad

Se evalúa si el punto óptimo analítico $(17.5, 10.5)$ respeta las limitantes de recursos físicas impuestas:

Horas Máquina: $2(17.5) + 3(10.5) = 66.5 \le 120 \quad \checkmark \text{ (Estable)}$

Materia Prima: $17.5 + 2(10.5) = 38.5 \le 80 \quad \checkmark \text{ (Estable)}$

Interpretación Final del Modelo

Dado que las coordenadas del vértice libre cumplen de manera holgada las inecuaciones de recursos, este se consolida como el óptimo definitivo.

Sustituyendo en la ecuación original: $Z^* = 35(17.5) - (17.5)^2 + 42(10.5) - 2(10.5)^2 = 526.75$

Óptimo Producto B10.5 u
